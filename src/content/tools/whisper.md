## 한줄 요약

OpenAI가 공개한 오픈소스 음성 인식(STT) 모델입니다. 한국어를 포함한 98개 언어를 인식하며, 로컬에서 무료로 실행할 수 있습니다.

## 무료 vs 유료

완전 무료 오픈소스입니다. 로컬 실행이 가능하고, OpenAI API를 통해서도 사용할 수 있습니다(API는 사용량 과금). 로컬 실행 시 Python 환경 설정이 필요합니다.

## 이런 분에게 추천

음성 데이터를 대량으로 텍스트 변환해야 하는 분, 자막 생성을 자동화하고 싶은 유튜버, 개발 프로젝트에 음성 인식을 통합하고 싶은 개발자에게 추천합니다.
