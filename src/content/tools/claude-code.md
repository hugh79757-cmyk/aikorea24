## 한줄 요약

터미널에서 Claude AI에게 복잡한 코딩 작업을 맡기는 도구입니다. 파일을 읽고, 수정하고, 테스트까지 AI가 자율적으로 수행합니다. 코딩 경험이 있는 개발자를 위한 고급 도구입니다.

## 이런 분에게 추천

이미 개발 경험이 있고, 반복적인 코딩 작업을 AI에게 위임하고 싶은 분에게 적합합니다. 초보자에게는 Cursor나 Lovable을 먼저 추천합니다.
