## 한줄 요약

GitHub와 OpenAI가 협력하여 만든 AI 코딩 어시스턴트입니다. VS Code 등 기존 에디터에 플러그인으로 설치해서 사용합니다. 코드를 작성하는 중간에 AI가 다음에 올 코드를 예측해서 자동완성해줍니다.

## Cursor와 차이점

Cursor는 대화형 코드 생성이 강점이고, Copilot은 코드 자동완성이 강점입니다. 이미 VS Code를 쓰고 있다면 Copilot을, 새로 시작한다면 Cursor를 추천합니다.
