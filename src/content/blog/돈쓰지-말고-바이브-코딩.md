## 필요한 도구는 딱 두 가지

**대화형 AI** 하나와 **VS Code** 만 있으면 됩니다. ChatGPT든 Claude든 젠스파크든 편한 걸 쓰시면 돼요. 젠스파크를 구독하면 여러 AI 모델을 한 곳에서 쓸 수 있어서 편리합니다.

[VS Code](https://code.visualstudio.com/download)는 무료 코드 편집기입니다. 여기에 터미널이라는 검은 화면이 있는데, 대부분의 작업이 여기서 이루어집니다. 처음엔 터미널이 무섭게 느껴질 수 있어요. 막상 써보면 그냥 명령어 입력하는 창입니다. AI가 이거 입력해봐 하고 알려주면 복사해서 붙여넣기만 하면 됩니다.

## 실제 비용은 얼마나 들까

AI 구독료와 도메인 비용 정도만 들어갑니다. 나머지는 전부 **무료 서비스** 로 가능해요. 코드 저장은 GitHub 무료 계정, 웹사이트 배포는 Cloudflare Pages나 Vercel 무료 플랜을 쓰면 됩니다.

제가 운영하는 모든 서비스가 이 구조입니다. 유료 구독은 젠스파크 1년 구독을 올해 초 할인할 때 구입했습니다. 서버 비용은 0원이에요. 무료로 시작해서 무료로 운영하는 게 충분히 가능합니다.

## 한번 배우면 점점 빨라집니다

처음엔 하나 만드는 데 며칠 걸렸습니다. 지금은 하루면 뼈대가 잡혀요. 한번 배운 게 다음에 또 쓰이니까 점점 빨라지더라고요. 매일 인스타그램과 쓰레드에 올리는 카드뉴스도 같은 방식으로 만들고 있습니다. 반복되는 작업은 자동화하고, 새로운 건 AI한테 물어보면서 해결하고. 이 사이클이 돌아가기 시작하면 할 수 있는 일이 계속 늘어납니다.

## 코딩 에이전트는 기본기 다음에

![Aj94hVA9.jpeg](https://img.aikorea24.kr/images/돈쓰지-말고-바이브-코딩/e3e57beef5fb03e81aea15e3ec07acd5b4799243.jpeg)

요즘 Claude Code나 Cursor 같은 코딩 에이전트들이 많이 나오고 있습니다. 화면에서 코드를 직접 보면서 고쳐주니까 정말 편리하죠. 하지만 처음 시작하는 분이라면 바로 이런 도구부터 쓰기보다는 기본적인 흐름을 먼저 이해하는 게 좋습니다.

코드가 어디에 저장되는지, 어떻게 웹사이트로 올라가는지, 에러가 나면 어떻게 해결하는지. 이런 기본을 알고 나면 어떤 에이전트가 나와도 원리가 보입니다. 새로운 도구가 나올 때마다 모두 사용할 필요는 없습니다.

참고로 Claude Code는 VScode의 터미널에서 이 한 줄이면 설치됩니다. 

npm install -g @anthropic-ai/claude-code

엔트로픽의 에이피아이를 사용하기 때문에 정기구독과는 또다른 방식으로, 사용한 만큼 비용이 차감됩니다. 우선 대화로 바이브코딩을 시작해보고 더 필요할 때, 에이젼트 서비스를 알아보는 것을 추천드립니다.




설치하고 프로젝트 폴더에서 claude라고 입력하면 바로 사용 가능합니다. 기본기를 익힌 다음에 쓰면 훨씬 효과적이에요.
