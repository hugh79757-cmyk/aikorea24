## MCP (Model Context Protocol)

AI 모델이 외부 도구·데이터에 접근하는 표준 프로토콜

### 상세 설명

Anthropic이 제안한 프로토콜로, AI가 파일 시스템, 데이터베이스, API 등 외부 리소스에 표준화된 방식으로 접근할 수 있게 합니다.

### 사용 예시

Claude가 MCP를 통해 구글 드라이브 파일을 읽는 것

### 관련 용어

에이전트, API, Claude
