## 에이전트 (Agent)

스스로 판단하고 도구를 사용해 작업을 수행하는 AI

### 상세 설명

단순히 질문에 답하는 것을 넘어, 웹 검색, 코드 실행, 파일 생성 등 도구를 활용해 복잡한 작업을 자율적으로 처리합니다.

### 사용 예시

Claude Code는 터미널에서 코드를 직접 작성하고 실행하는 에이전트입니다.
