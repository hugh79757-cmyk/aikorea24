## 도구 사용 (Tool Use)

AI가 계산기, 검색 등 외부 도구를 호출해 사용하는 기능

### 상세 설명

Function Calling이라고도 합니다. AI가 스스로 필요한 도구를 선택해 호출하고 결과를 활용합니다.

### 사용 예시

환율 계산이 필요하면 환율 API를 호출해서 정확한 값을 제공
