## 오케스트레이션 (Orchestration)

여러 AI 모델과 도구를 조합해 복잡한 작업을 수행하는 것

### 상세 설명

하나의 AI가 아니라 여러 AI와 도구를 연결하여 복잡한 워크플로우를 자동으로 처리하는 패턴입니다.

### 사용 예시

고객 문의 → 감정 분석 AI → 카테고리 분류 AI → 답변 생성 AI

### 관련 용어

에이전트, 워크플로우, 자동화
