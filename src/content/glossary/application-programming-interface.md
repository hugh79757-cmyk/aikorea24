## API (Application Programming Interface)

프로그램끼리 소통하는 약속된 방법

### 상세 설명

AI 서비스를 웹사이트나 앱에 연결할 때 사용합니다. API를 통해 내 앱에서 ChatGPT나 Claude의 기능을 호출할 수 있습니다.

### 사용 예시

OpenAI API를 이용하면 내 웹사이트에 AI 챗봇을 넣을 수 있습니다.

### 관련 용어

SDK, 엔드포인트, REST
