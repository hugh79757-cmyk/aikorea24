## 바이브 코딩 (Vibe Coding)

AI에게 자연어로 설명해서 코드를 만드는 방식

### 상세 설명

코딩 문법을 몰라도 원하는 기능을 말로 설명하면 AI가 코드를 생성해줍니다. Lovable, Bolt, Cursor 등이 대표적입니다.

### 사용 예시

'로그인 페이지 만들어줘'라고 입력하면 AI가 코드를 생성

### 관련 용어

Lovable, Cursor, Bolt, 노코드
